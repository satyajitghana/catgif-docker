# catgif

Go to <https://labs.play-with-docker.com/>

## Clone

```bash
git clone https://github.com/satyajitghana/catgif-docker
cd catgif-docker
git checkout compose
```

## Build

```bash
docker compose build
```

## Run

```bash
docker compose up
```

Now Open Port `3000` of the Machine (Play With Docker)
